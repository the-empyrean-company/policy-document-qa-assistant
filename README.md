# Policy / Document Q&A Assistant

A retrieval-augmented AI assistant for querying policy and knowledge documents using natural language.

## Features
- Upload PDF and TXT documents
- Ask questions grounded in document content
- Source-aware answers
- Simple web UI built with Gradio

## Tech Stack
- Python
- Gradio
- OpenAI API
- FAISS (planned)

## Use Cases
- Government policy exploration
- Internal corporate knowledge bases
- Consulting and compliance teams

## Status
This project is under active development.
